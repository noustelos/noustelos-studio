System Notes

Το setup μένει κοντά στο original code idea, αλλά προσαρμόζεται σε standalone page χωρίς εξωτερικά imports, με responsive rebuild και touch-friendly pointer interaction.

01 / mesh

Code cloth

Οι χαρακτήρες γίνονται particles με constraints ώστε να σχηματίζουν εύκαμπτο grid που λυγίζει και επιστρέφει οργανικά.

canvas physics glyphs
02 / input

Pointer field

Το pointer μπορεί είτε να τραβήξει άμεσα το mesh είτε να το διαταράξει με ήπια force field κίνηση γύρω από τα glyphs.

drag touch repel
