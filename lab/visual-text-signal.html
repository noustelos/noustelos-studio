CSS Notes

Το effect μένει κοντά στο original snippet αλλά γίνεται production-friendly με standalone page framing, browser fallback όταν δεν υπάρχει scroll timeline support και πιο καθαρό visual hierarchy για το lab.

01 / reveal

View timeline

Η αποκάλυψη του κειμένου τρέχει με pure CSS view timeline ώστε κάθε heading να γεμίζει όσο μπαίνει στο viewport.

02 / hover

Line inversion

Το overlay span ανοίγει σε ολόκληρη τη γραμμή στο hover, μετατρέποντας το static heading σε interactive signal strip.

03 / fallback

Graceful support

Όπου δεν υποστηρίζεται το animation timeline, το text μένει πλήρως readable αντί να φαίνεται ημιτελές ή κρυμμένο.
