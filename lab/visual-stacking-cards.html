Card Stack

  • 01

    Sticky Stacking

    Utilizes position: sticky to create a stacking context where elements layer on top of one another naturally within the flow.

  • 02

    Scroll Driven

    Animations are powered by animation-timeline: view(), allowing precise control based on element visibility in the viewport.

  • 03

    3D Depth

    Combines perspective with 3D transforms to create depth perception, tilting cards backward as they ascend the scroll port.

  • 04

    Performant

    Animations run on the compositor thread for smooth 60fps motion without layout thrashing and main-thread blocking.
