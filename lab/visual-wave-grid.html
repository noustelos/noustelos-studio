Canvas Field

Wave Grid

Fullscreen geometric grid που αντιδρά σε pointer proximity και click-triggered waves, με masked overlays ώστε το UI να μοιάζει κομμένο μέσα από το reactive pattern.

Grid reactive shapes
Click wave propagation
Mask ui cutout logic
01 / motion

Pointer bloom

Τα shapes μεγαλώνουν με smooth falloff γύρω από το pointer και κάθε hover cycle ανανεώνει scale, rotation και star geometry.

02 / waves

Click shock

Κάθε click παράγει wave ring που ταξιδεύει πάνω στο grid και παρακάμπτει προσωρινά τα masked panels για να φαίνεται καθαρά η διάδοση.

03 / palette

Gradient mix

Solid και radial gradient fills εναλλάσσονται τυχαία ώστε το πεδίο να παραμένει ζωντανό χωρίς να μοιάζει επαναλαμβανόμενο.

Interaction

Κινήσου πάνω στο πεδίο για local activation και κάνε click οπουδήποτε για wave burst. Τα overlay blocks χρησιμοποιούν το attribute data-shape-mask για να μένουν καθαρά.
