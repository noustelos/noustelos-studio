Water Cycle System

High-Performance Pool & Spa Landing Platform — a production-grade, bilingual (EN/EL) marketing and lead-generation platform for a premium pool and spa service provider in Santorini, balancing serverless agility with robust backend security.

Overview

A production-grade, bilingual (EN/EL) marketing and lead-generation platform for a premium pool and spa service provider in Santorini. Built as a high-performance static-first ecosystem, balancing serverless agility with robust backend security.

Core Stack

Node.js · Express · Vanilla JS (ES Modules) · Tailwind CSS · Resend API · Hugging Face / Gradio.

Technical Highlights

  • Frontend — Zero-framework SPA architecture using ES Modules for sub-second Time-to-Interactive (TTI) on mobile. Includes a custom i18n engine with localStorage persistence and orthographic-aware Greek typography.
  • AI Integration — Embedded Gradio AI chat assistant, optimized for low latency via a visibility-aware keep-warm ping strategy and deferred loading.
  • Backend — Hardened Express API featuring strict security middleware (HSTS, X-Content-Type-Options), in-memory rate limiting (sliding window), and robust input sanitization against injection attacks.
  • DevOps — Automated CI/CD pipeline via GitHub Actions. Supports dual deployment: static hosting via GitHub Pages and high-availability VPS bootstrap (Ubuntu + PM2 + Nginx + UFW).
