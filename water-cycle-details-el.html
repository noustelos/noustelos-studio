Πλατφόρμα Υπηρεσιών Pool & Spa — μια πλήρως λειτουργική, δίγλωσση (EN/EL) πλατφόρμα marketing και lead-generation για επιχείρηση συντήρησης πισίνας και spa στη Σαντορίνη, που συνδυάζει static-first αρχιτεκτονική με ένα ισχυρό backend.

Επισκόπηση

Μια πλήρως λειτουργική, δίγλωσση (EN/EL) πλατφόρμα marketing και lead-generation για επιχείρηση συντήρησης πισίνας και spa στη Σαντορίνη. Το project εστιάζει στην απόλυτη ταχύτητα και ασφάλεια, συνδυάζοντας static-first αρχιτεκτονική με ένα ισχυρό backend.

Core Stack

Node.js · Express · Vanilla JS (ES Modules) · Tailwind CSS · Resend API · Hugging Face / Gradio.

Τεχνικά Χαρακτηριστικά

  • Frontend — Zero-framework SPA αρχιτεκτονική για άμεση απόκριση σε mobile συσκευές. Περιλαμβάνει custom i18n engine για εναλλαγή γλωσσών και προσαρμοσμένη τυπογραφία για σωστά ελληνικά (stripping tonos).
  • AI Integration — Ενσωματωμένο Gradio AI chat, βελτιστοποιημένο με keep-warm στρατηγική για εξάλειψη του πρώτου latency.
  • Backend — Θωρακισμένο Express API με security headers, in-memory rate limiting κατά των spam και αυστηρό input validation.
  • DevOps — Πλήρως αυτοματοποιημένο CI/CD (GitHub Actions). Υποστηρίζει dual deployment: GitHub Pages για το static κομμάτι και αυτοματοποιημένο VPS bootstrap (Ubuntu, PM2, Nginx, UFW) για το backend.
